## 📌 О проекте

Это **полностью рабочий Telegram‑бот**, который автоматически:
- проводит пользователя по воронке из 3 шагов (уровень → цель → время),
- персонализирует предложение (разный ответ для разных типов клиентов),
- принимает оплату через ЮKassa,
- отправляет лиды в CRM (Битрикс24).

> Проект создан как портфолио — чтобы показать навыки работы с логикой ботов, переменными, внешними API и платёжными системами. При этом он полностью готов к бизнес‑задачам.

## 🎯 Что даёт бизнесу

1. **Квалификация лидов** без участия менеджера.
2. **Автоматический подбор тарифа** на основе ответов.
3. **Онлайн‑оплата** внутри Telegram.
4. **CRM в подарок** — все заявки сразу в Битрикс24.

## 🛠️ Стек технологий

| Сервис | Роль |
|--------|------|
| **Salebot.pro** | Платформа для бота (логика, переменные, переходы) |
| **ЮKassa** | Приём платежей (тестовый режим + обработка коллбэков) |
| **Битрикс24** | CRM (создание лидов через REST API / вебхуки) |
| **Telegram** | Канал коммуникации |
| **ИИ** | Генерация текстов сценария и обложки для портфолио |

## ✨ Мои решения и функционал

### 🔁 Разветвлённая логика
- Использованы переменные `level`, `goal`, `time`.
- Для каждой комбинации (Новичок+Карьера, Опытный+Проект и т.д.) — свой финальный ответ.
- Добавлена **«стрелка по умолчанию»** (else‑условие) — обрабатывает любые другие комбинации, бот никогда не зависает.

### 💳 Платежи
- Подключена ЮKassa через Salebot.
- Реализована обработка **асинхронного коллбэка** — бот узнаёт об успешной оплате в реальном времени и переводит пользователя на финальный шаг.

### 🏢 Интеграция с CRM
- Настроен **входящий вебхук** Битрикс24.
- Метод `crm.lead.add` автоматически создаёт лид с именем, телефоном и комментарием по ответам пользователя.

### 🧪 Стабильность
- Промежуточный блок «Ожидание оплаты» гарантирует корректную обработку коллбэка.
- Есть отдельный блок для ошибок оплаты / отмены.
